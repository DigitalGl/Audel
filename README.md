# Audel

Audel is a local desktop music player for Windows, macOS, and Linux.

## Features

- Local audio library and folder scanning.
- Manual playlist ordering.
- Custom covers for tracks and playlists.
- Fullscreen now-playing view with queue toggle.
- Waveform or classic line progress display.
- Optional internet download tools powered by bundled helper binaries.

## Development

Install dependencies and run the app:

```powershell
npm install
npm start
```

Build a desktop package:

```powershell
npm run dist
```
